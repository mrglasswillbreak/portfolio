# 🌟 mrglasswillbreak Portfolio

This project is a Next.js portfolio site now customized to use **mrglasswillbreak** profile details and links.

## 🔗 Profile
- GitHub: https://github.com/mrglasswillbreak

## 🚀 Run locally
```bash
npm install
npm run dev
```

Then open http://localhost:3000.

## ⚙️ Main profile config
Update personal/site data in:
- `src/constant/self.ts`
- `src/app/layout.tsx`
- `src/lib/structured-data.ts`

## 📝 Notes
- Resume viewer currently uses `public/docs/Aarab_Nishchal_Resume.pdf`.
- Social links are generated from usernames in `selfData.socials_username`.
